\# Model Plan: PhishShield AI



\## Purpose of This Document



This document defines the machine learning and AI strategy for PhishShield AI.



It explains which models will be built, in what order they will be built, what each model is responsible for, and how model performance will be evaluated.



The goal is to avoid jumping directly into complex models without first creating strong baselines.



\---



\## Model Development Philosophy



The project will follow this order:



```text

Rule-Based Baseline
