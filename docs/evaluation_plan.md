

\# Evaluation Plan: PhishShield AI



\## Purpose of This Document



This document defines how PhishShield AI will be evaluated.



The goal is to measure whether the system is actually useful, not just whether it gives outputs.



Phishing detection must be evaluated carefully because both false positives and false negatives matter.



\---



\# Evaluation Layers



The project will be evaluated in five layers:



1\. Phishing classification performance

2\. Intent detection performance

3\. Language-specific performance

4\. Behavioral anomaly scoring usefulness

5\. System performance
